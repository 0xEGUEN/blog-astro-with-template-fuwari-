## Hello World

Kalau kamu baru belajar pemrograman, hampir pasti kamu pernah nulis ini:

```python
print("Hello, World!")
```

Atau ini:

```javascript
console.log("Hello, World!");
```

Atau bahkan ini:

```c
#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}
```

## Kenapa Hello World?

**Hello World** adalah program pertama yang biasanya ditulis saat belajar bahasa pemrograman baru. Tujuannya sederhana: **memastikan lingkungan development kamu berfungsi dengan benar**.

Dengan mencetak teks "Hello, World!" ke layar, kamu sudah melakukan beberapa hal penting:

1. **Menulis kode** — sintaks dasar bahasa tersebut
2. **Menjalankan kode** — menggunakan compiler atau interpreter
3. **Melihat output** — program berhasil berjalan

## Filosofi di Baliknya

Hello World mengajarkan prinsip paling dasar dalam programming:

> **Input → Proses → Output**

Sesederhana apapun programnya, konsep ini selalu berlaku. Kamu memberi instruksi ke komputer, komputer memprosesnya, lalu memberikan hasil.

## Bahasa Pemrograman Lain

```java

    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

```go

import "fmt"

    fmt.Println("Hello, World!")
}
```

```rust

    println!("Hello, World!");
}
```

```ruby
puts "Hello, World!"
```

```php
<?php
echo "Hello, World!";
?>
```

```kotlin
fun main() {
    println("Hello, World!")
}
```

## Dari Sini Semua Dimulai

Hello World mungkin terlihat sepele, tapi itu adalah **langkah pertama** yang pernah diambil oleh setiap programmer di dunia. Dari sini, kamu bisa lanjut ke variabel, perulangan, fungsi, dan seterusnya.

:::tip
Setiap kali belajar bahasa baru, selalu mulai dengan Hello World. Itu cara tercepat untuk tahu "apa ini bekerja?".
:::

Jadi, kalau kamu baru mulai — **selamat datang!** Kamu udah ambil langkah pertama.
